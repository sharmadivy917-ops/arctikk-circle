# Arctikk Circle — Course Sales Platform

Live site: https://arctikkcircle.in

## What is this?
During my 1-month internship I worked on a full course selling 
platform for a client from scratch. Not just the UI — I handled 
payments, emails, tracking, data and even built a small dashboard 
plugin so the client could manage everything without touching code.

## What I built

- Landing page designed to convert visitors into buyers
- OTO (one-time offer) page as part of the sales funnel
- Video page where enrolled students access the course
- A custom WordPress plugin with a UI dashboard for the 
  client to track sales and manage customer data

## Integrations I handled

- Razorpay — so customers can pay directly on the site
- Omnisend — automatic emails triggered after purchase
- Google Sheets — every sale gets backed up as a spreadsheet row
- Google Tag Manager — set up event tracking across all pages
- Meta Pixel — connected for Facebook and Instagram ad tracking
- Meta Conversion API — server side tracking so no data is lost 
  even when browser blocks the pixel

## Tech used
HTML, CSS, JavaScript, WordPress, Elementor, Hostinger
